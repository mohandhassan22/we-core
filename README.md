# WE-Core | المنصة المتكاملة لفريق العمل

## نبذة عن المشروع

WE-Core هي منصة متكاملة تهدف إلى تسهيل بيئة العمل وجمع كل ما يحتاجه فريق العمل من فورمات، معلومات، وعروض في مكان واحد منظم وسهل الوصول. تهدف المنصة إلى توفير الوقت وتقليل الجهد من خلال إتاحة البيانات والمعلومات بضغطة زر واحدة. [1]

## الميزات الرئيسية

تقدم منصة WE-Core مجموعة من الأقسام الرئيسية لتلبية احتياجات فريق العمل:

*   **All PDF**: جميع ملفات PDF الخاصة بالخدمات والعروض.
*   **All Info**: معلومات شاملة حول خدمات Mobile و ADSL و Fixed.
*   **Remedy**: قسم مخصص لجميع التذاكر (Tickets) المتعلقة بالموبايل.
*   **Courses**: دروس تعليمية ودورات تدريبية مثل BSS Tutorial.
*   **Offers**: جميع العروض المتاحة والباقات الحالية.
*   **WE Pay**: معلومات حول محفظة WE Pay وخدمات الدفع الإلكتروني.
*   **Info Games**: معلومات حول الألعاب والحزم المتوفرة.

## التقنيات المستخدمة

تم تطوير هذه المنصة باستخدام التقنيات التالية:

*   **HTML**: لبناء هيكل صفحات الويب.
*   **CSS**: لتصميم وتنسيق الواجهة الرسومية.
*   **JavaScript**: لإضافة التفاعل والوظائف الديناميكية.
